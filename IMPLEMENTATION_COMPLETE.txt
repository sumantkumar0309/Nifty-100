DEPLOYMENT STEPS
================================================================================

LOCAL DEVELOPMENT:
  1. cp .env.example .env
  2. pip install -r requirements.txt
  3. python manage.py migrate
  4. redis-server &
  5. celery -A backend2 worker -l info &
  6. celery -A backend2 beat -l info &
  7. python manage.py runserver

PRODUCTION (DOCKER):
  1. Generate SSL certificates
  2. Update .env with production values
  3. Configure GitHub secrets (DOCKER_USERNAME, DEPLOY_KEY, etc.)
  4. docker-compose -f docker-compose.prod.yml build
  5. docker-compose -f docker-compose.prod.yml up -d
  6. docker-compose -f docker-compose.prod.yml exec django python manage.py migrate
  7. Verify all services are healthy: docker-compose ps
